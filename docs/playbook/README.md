# Repo Playbook: Procedural Memory for AI Agents

This directory contains distilled, reusable procedures discovered during past
agent sessions. Unlike general documentation, the Playbook focus on "how to
solve" recurring technical challenges in this specific repository.

## Structure

- **Procedures**: Step-by-step guides for complex multi-step tasks.
- **Anti-Patterns**: What NOT to do, based on past failures.
- **Discovery Log**: Raw technical "nuggets" that haven't been distilled yet.

## Pipeline

1. **Episodic**: Use `cass` to find raw session transcripts.
2. **Reflective**: Distill findings into a new entry here.
3. **Curative**: Refine existing entries as the repo evolves.
